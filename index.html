สาริน ขุนจันทร์

Software Engineer, Frontend

ผมนาย สาริน ขุนจันทร์ อายุ 21ปี ปัจจุบันกำลังศึกษาอยู่ชั้นปีที่ 3 สาขาสารสนเทศทางธุรกิจ คณะบริหารธุรกิจ มหาวิทยาลัยเทคโนโลยีราชมงคลศรีวิชัย ในระหว่างที่เรียนกระผมได้ฝึกและได้รับงานทางด้าน Front-end ที่ได้ใช้ทั้ง html, React, Mongodb และ Firebase รวมทั้งการวาดคอนเซปฮาร์ต ในการจัดกิจกรรมต่างๆ ของทางมหาวิทยาลัย นอกจากนี้กระผมยังได้เป็นสมาชิกองค์การนักศึกษาประจำปีการศึกษา 2563 ทำให้ได้มีโอกาสเรียนรู้การบริหาร การวางแผน การทำงานเป็นทีม การคิดแก้ปัญหาและการตัดสินใจ

ประวัติการศึกษา

  1. ระดับประถมศึกษา

    โรงเรียนสาธิตมหาวิทยาลัยราชภัฏสงขลา

    โรงเรียนที่สอนในระดับประถมศึกษาตั้งอยู่ในมหาวิทยาลัยราชภัฏสงขลาจังหวัดสงขลา

  2. ระดับมัธยมศึกษาตอนต้นและปลาย

    โรงเรียนวรนารีเฉลิม แผนกศิลป์คํานวณ

    โรงเรียนประจำจังหวัดตั้งอยู่ในจังหวัดสงขลา ผมได้ศึกษาแผนกศิลป์คำนวณได้ศึกษาเกียวกับเลขด้านคำนวณและภาษาต่างประเทศ ตลอดเวลาที่ผมได้เรียนระดับชั้นมัธยมศึกษา ผมได้ศึกษาและได้เรียนทางด้านคอมพิวเตอร์เช่น ภาษา c++ และ c# รวมถึงการเรียนโครงสร้างข้อมูลและเทคโนโลยีสารสนเทศ

  3. ระดับปริญญาตรี

    ระบบสารสนเทศ

    สาขานี้ศึกษาเกี่ยวกับพื้นฐานระบบสารสนเทศ การโปรแกรมเชิงตรรกะและโครงสร้างข้อมูล การเขียนโปรแกรมคอมพิวเตอร์ การสื่อสารข้อมูลและเครือข่ายคอมพิวเตอร์ การวิเคราะห์และออกแบบระบบ การบริหารโครงการทางเทคโนโลยีสารสนเทศ ระบบสารสนเทศเพื่อการจัดการ
